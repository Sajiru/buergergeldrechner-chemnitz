#### Allgmeine Hinweise

Diese Seite zeigt das Ergebnis Ihres möglichen Anspruchs an.

Aus diesem Ergebnis können keine Rechtsansprüche abgeleitet werden.

Verfügen Sie über niedriges Einkommen, dann können Sie Wohngeld beantragen.

Verfügen Sie über niedriges Einkommen und haben Kinder in Ihrer Bedarfsgemeinschaft können Sie Kinderzuschlag beantragen.

Sollten Sie den Anspruch auf Bürgergeld prüfen lassen wollen, dann empfehlen wir Ihnen dies Online über www.jobcenter.digital zu tun. Halten Sie dafür alle erforderlichen Nachweise bereit. Nur so können Sie eine zügige Antragsbearbeitung unterstützen.
